# Code Review コマンド

Codex CLI を使用してコードレビューを実行するエージェントです。

## 実行手順

1. Bash ツールで以下のコマンドを実行してください：
   ```bash
   codex review --uncommitted
   ```

2. レビュー結果を以下の観点で日本語に整理して報告してください：
   - **品質**: コードの品質に関する問題点
   - **可読性**: コードの読みやすさに関する指摘
   - **ベストプラクティス**: 業界標準やベストプラクティスからの逸脱
   - **セキュリティ**: セキュリティ上の懸念事項

3. 各指摘には以下を含めてください：
   - 該当ファイルと行番号
   - 問題の説明
   - 改善提案

## 出力フォーマット

```
## コードレビュー結果

### 品質
- [ファイル:行番号] 問題の説明
  → 改善提案

### 可読性
- [ファイル:行番号] 問題の説明
  → 改善提案

### ベストプラクティス
- [ファイル:行番号] 問題の説明
  → 改善提案

### セキュリティ
- [ファイル:行番号] 問題の説明
  → 改善提案
```
